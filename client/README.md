# 🏠 PG Buddy

[PG Buddy](https://github.com/calvintakasi/PG-Buddy) helps students and professionals easily find Paying Guest (PG) accommodations and hostels. 
It provides verified listings, detailed PG information, and an easy-to-use interface for browsing, booking, and managing PGs.

## 🚀 Features  

- 🔍 **Browse PGs** – Explore verified listings with details and images.  
- 📄 **PG Details** – View full details of any PG/Hostel.  
- 🔑 **User Authentication** – Login & Signup functionality.  
- 📱 **Responsive UI** – Works seamlessly across devices.  
- 🧭 **How It Works Section** – Guides new users through the process.  
- 🖼️ **Modern UI** – Built with React + TailwindCSS.  
- 📌 **Navigation** – Easy routing using React Router.  
- 📬 **Footer with Contact Info & Social Links**
